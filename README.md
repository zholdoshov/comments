# Comments CRUD App

Simple full-stack CRUD application for managing comments.

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: SQLite (better-sqlite3)

## Setup

```bash
# Backend
cd backend
npm install

# Frontend
cd frontend
npm install
```

## Run

```bash
# Backend
npm run dev

# Frontend
npm run dev
```

Backend runs on http://localhost:3005, frontend on http://localhost:5173.
